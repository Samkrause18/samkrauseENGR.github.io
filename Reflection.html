Reflection

Overall, I found the process really fun because I had never really done any 3D printing and it is really cool just to be able to make something out of nothing. It was interesting testing the printer to failure because it is not something I would have thought we would do, but I learned a lot of valuable information. Something I will do next time is be more conscious about the filament color and size of the model, making sure it is exactly what I want.
